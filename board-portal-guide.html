Overview

Welcome to the Board Portal

The board portal has two main areas: the Internal Messenger for board conversation and coordination, and the Document Sections for storing official files. Both are accessible from the sidebar navigation on the left side of the board page.

⚠️

Sensitive documents must never be stored in the Messenger. Contracts, financials, legal filings, research proposals, ethics documentation, and other official records belong in the appropriate folder section. The Messenger is for discussion only.

Navigation

Navigating the Portal

The left sidebar shows all sections of the board portal. Click any section name to jump directly to it. The active section is highlighted. Available sections include: Messenger, Finance, Fundraising, Governance, Marketing, Research, Meetings, Networking, Operations, and the Board Tracker for action items.

Internal Messenger

Internal Messenger

The Messenger is a real-time communication tool for board members. Messages are organized by topic and categorized by type. Use it to discuss ideas, flag decisions that need input, assign action items, and share updates.

Topics

Every message belongs to a topic. Use the sidebar to switch between topics. Announcements and General appear at the top of the list, followed by topic-specific channels. When you navigate to a topic, the compose area will automatically set that topic on your next message.

All Messages Announcements General Finance Fundraising Governance Marketing Research & Partnerships Meetings Networking Operations

If you're unsure which topic to use, post to General. You can always refile a message into the correct topic later.

Message Types

Every message has a type that signals what kind of response — if any — is expected. Choose the type that matches your intent when composing.

Idea / Brainstorm

Thoughts, suggestions, or early discussion. No action or decision required — just something worth sharing.

Decision Needed

The board hasn't decided yet. Use when you need input, a vote, or a clear direction from the group.

Action Item

The decision is already made. Use when there's a specific task, a clear owner, and a next step to complete.

Update

Progress reports, status updates, or completed work. Keeps the board informed without requiring a response.

ℹ️

Decision Needed vs. Action Item: If you're still asking "what should we do?" — that's Decision Needed. If the decision is made and something needs to happen — that's an Action Item.

Flags

Flags add urgency or accountability signals to a message. They are optional — use them when genuinely needed.

Reply Needed Urgent

Reply Needed signals that you are waiting on a response before moving forward. Urgent signals that something is time-sensitive. Use both sparingly so they retain meaning.

Sending a Message

1

Click inside the compose area at the bottom of the Messenger. It will expand to show all options.

2

Type your message in the text field.

3

Confirm or change the Topic in the dropdown. If you navigated to a topic in the sidebar, it will already be selected.

4

Choose a Type — Idea, Decision Needed, Action Item, or Update.

5

Optionally toggle Reply Needed or Urgent if appropriate.

6

Optionally attach a file using the attachment button. For official documents, use the document folder sections instead.

7

Click Send. Your message appears immediately.

To cancel without sending, click the ✕ button to collapse the compose area.

Replying to a Message

Hover over any message to reveal action buttons. Click Reply to open a reply box directly beneath that message. Replies stay connected to the original message regardless of which topic filter is active.

Refiling a Message

If a message was sent to the wrong topic or given the wrong type, use Refile to correct it without deleting and reposting.

1

Hover over the message to reveal action buttons.

2

Click Refile. A panel will appear below the message.

3

Select the correct Topic and Type from the dropdowns.

4

Click Save. The message moves to the new topic immediately.

Deleting a Message

Hover over any message to reveal the Delete button. A confirmation prompt will appear before the message is permanently removed. Deletions cannot be undone.

To remove only a file attachment while keeping the message text, click the small ✕ next to the file link. The text is preserved; only the attachment is removed.

⚠️

If you only need to fix the topic or type, use Refile instead of deleting. Refile preserves the message and any replies attached to it.

Filtering and Sorting

The filter bar at the top of the Messenger lets you narrow messages by type: All, Action Items, Decisions Needed, Urgent, or Assigned to Me. These filters work within whatever topic is currently selected in the sidebar.

Use the Oldest first / Newest first toggle button in the filter bar to change the sort order. The setting persists as you browse between topics.

Document Sections

Document Sections

Each section of the board portal has its own document library organized into folders. Files stored here are accessible to all board members at any time. This is where official, permanent records should live — not in the Messenger.

Uploading a Document

1

Navigate to the correct section in the sidebar (Finance, Governance, Research, etc.).

2

Click the Upload Document button near the top of that section.

3

In the upload panel, confirm the Section and choose the correct Subfolder.

4

Enter a clear document title and the date of the document.

5

Click Choose File and select the file from your computer.

6

Click Save Document. The file appears in the correct folder immediately.

Important Rules

Important Rules

Keep Sensitive Documents Out of the Messenger

The Messenger is designed for discussion and coordination — not for storing official records. Anything that is a formal document should be uploaded to the appropriate folder section, not attached to a chat message.

This includes: contracts, legal filings, financial statements, tax documents, research proposals, ethics documentation, grant agreements, insurance policies, HR records, and governance documents.

🔒

If you're unsure whether something belongs in the Messenger or a folder section, ask yourself: Is this a record the foundation needs to keep? If yes — it goes in a folder section, not the chat.

Use the Right Type

Selecting the correct message type helps the whole board quickly understand what is needed from them. A message marked Decision Needed signals that input is required. An Action Item signals that something needs to be done. Using these correctly reduces confusion and keeps work moving.

Refile Instead of Delete

If a message was posted to the wrong topic or given the wrong type, use Refile to correct it rather than deleting and reposting. Refile preserves the message history and any replies that have been added to it.

Use Flags Sparingly

Flags only work when they are meaningful. If every message is marked Urgent, none of them are. Reserve Urgent for genuinely time-sensitive situations, and Reply Needed for cases where you truly need a response before moving forward.
