Page Elements — Technical Names

Use the name in the Element column when asking for changes. The Selector column shows the exact CSS name used in the code.

Element Selector Description
Navigation Bar
Nav bar #site-nav The sticky header bar that stays at the top of the page as you scroll. Contains the logo and navigation links.
Nav background #site-nav { background } The background color of the entire nav bar. Currently #fbf9f6 (warm off-white).
Nav bottom border #site-nav { border-bottom } The thin line separating the nav from the page content below it.
Logo area .nav-logo The clickable logo group containing the logo image and the foundation name/tagline text. Clicking it returns to the home page.
Logo image .logo-svg The circular logo PNG image in the nav bar. Currently 88×88px.
Foundation name .nav-logo-text .name "SETD5 Syndrome Foundation" — the large text next to the logo. Currently 2rem, weight 700, teal color.
Nav tagline .nav-logo-text .tagline "research · advocacy · community" — the smaller text beneath the foundation name. Coral color, condensed font.
Nav links .nav-links a The clickable page links in the nav bar (About, Research, Resources, Donate). Currently 1rem, weight 600.
Donate button (nav) .nav-donate The coral-filled "Donate" button in the nav bar. Styled differently from the other nav links.
Dropdown menu .nav-dropdown-menu The pop-out list that appears when hovering over "About" in the nav. Contains sub-page links.
Dropdown caret .nav-caret The small ▾ arrow next to "About" that opens the dropdown menu when clicked.
Page Layout
Container .container The invisible box that keeps content centered and limits it to 1200px wide with 24px padding on each side. All body content sits inside this.
Narrow container .container--narrow A slimmer version of the container, max 780px wide. Used for text-heavy pages like Resources.
Section .section A full-width horizontal band that groups related content. Sections stack vertically down the page and alternate background colors.
Page banner .page-banner The teal header strip at the top of inner pages (Research, Resources, etc.) that shows the page title and a short description.
Page banner title .page-banner h1 The large white heading inside the teal page banner.
Page banner subtitle .page-banner p The smaller descriptive text below the page banner title.
Home Page — Hero
Hero section .hero The large opening section on the home page with the headline, description, buttons, and stats card.
Hero headline .hero-content h1 The large bold text at the top of the hero ("Turning Research Into Hope...").
Hero description .hero-desc The paragraph text beneath the hero headline describing the foundation's mission.
Hero buttons .hero-actions The row of buttons in the hero (e.g. "Join Our Community", "Learn About SETD5 Syndrome").
Hero stats card .hero-card The floating card on the right side of the hero that shows key statistics.
Hero stat .hero-stat An individual number + label pair inside the hero stats card (e.g. "1 in 30,000 births").
Hero quote .hero-tagline The italic quote at the bottom of the hero stats card ("Every dollar brings us closer to answers.").
Home Page — Other Sections
Impact stats bar .stats-bar The full-width band beneath the hero showing 3–4 high-level numbers. Outer background: #cfcfcc with drop shadow. Inner panel (.stats-bar-inner) has a rounded #FAFAF8 inset with a soft inner shadow.
Stats bar inner panel .stats-bar-inner The rounded #FAFAF8 container that holds the stat items inside the stats bar. Has border-radius: 14px and an inset box-shadow.
Stat item .stat-item An individual number + label pair inside the stats bar. Number (.num) uses --teal at 2rem; label (.lbl) uses --text-mid at 0.68rem uppercase.
Stat divider .stat-divider A 1px vertical separator between stat items. Soft teal tint: rgba(47,83,97,0.15). Hidden on mobile.
What is section .what-is (section) The "What is SETD5 Syndrome?" explainer section on the home page with the symptom cards.
Story section .story-card The family quote/testimonial section on the home page with the pull quote and attribution.
CTA band .cta-band A full-width teal call-to-action strip that appears at the bottom of pages with a headline and buttons encouraging action.
Newsletter section .newsletter The email sign-up section with the input field and subscribe button.
Components
Card .card A white rounded box with a subtle shadow used to display a discrete piece of content (research highlight, symptom, feature, etc.).
Card icon .card-icon The small 52×52px square tile with an icon at the top of a card. Uses --coral-light background (#eeebe7) with a #c96a38 stroke. SVG icons are 22×22px.
Eyebrow / subheading .subhead The small all-caps coral label that appears above a section heading (e.g. "OUR MISSION", "ABOUT THE SCIENCE").
Primary button .btn-primary A coral-filled button. Used for the most important action on any given section.
Outline button .btn-outline A teal-bordered transparent button. Used for secondary actions on light backgrounds.
Gold button .btn-gold A gold-filled button. Used for fundraising emphasis.
Secondary button .btn-secondary A white-bordered transparent button. Used on dark teal backgrounds only.
Large button .btn-lg An extra-large button variant. Added alongside any button class (e.g. .btn-primary.btn-lg) for hero CTAs.
About the Foundation Page (Leadership)
Board grid .board-grid The 3-column grid showing the 5 board members with their photo, name, title, and bio.
Board member card .board-card An individual board member's card inside the board grid.
Board member photo .board-photo The circular headshot image on a board member card.
Board member name .board-card h3 The name heading on a board member card.
Board member title .board-title The role/title line beneath the name on a board member card (e.g. "Founder & Executive Director").
Working group section .working-groups (section) The section listing the foundation's working groups with their focus areas.
Footer
Footer #site-footer The dark teal section at the very bottom of every page with navigation links, contact info, and the copyright line.
Footer logo .footer-logo The foundation name and tagline repeated in the footer.
Footer columns .footer-grid The multi-column layout inside the footer with grouped links.
Footer copyright .footer-bottom The bottom strip of the footer with the copyright notice and legal links.

Components

Buttons

All buttons use border-radius: 6px, font-weight: 700, and a 2px border for consistent shape across variants.

Donate Now .btn-primary — coral fill, used for primary CTAs
Learn More .btn-outline — teal border, used for secondary actions on light backgrounds
Support Research .btn-gold — gold fill, used for emphasis / fundraising accents
Large Button .btn-lg — larger padding for hero CTAs
Get Involved .btn-secondary — white border, for use on teal backgrounds only

Components

Cards

Standard Card

Terracotta tint icon background (--terra-light) with #c96a38 stroke. Used throughout the site for features, actions, and community content.

Card with Icon

All card icons are unified — single terracotta treatment across all card types. SVGs are 22×22px within a 52×52px tile.

Hover Lift

All cards lift on hover: translateY(-3px) + --shadow. Base shadow is --shadow-sm.

Cards use border-radius: 12px, box-shadow: --shadow-sm, and lift on hover via translateY(-3px). Icon tile: --coral-light background (#eeebe7), #c96a38 stroke, 22×22px SVG.
